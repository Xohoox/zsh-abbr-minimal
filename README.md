# zsh-abbr-minimal
A minimal abbriviations plugin for zsh. Use Space to expand the abbreviation and Ctrl+Space if you don't want it to expand.

## Configuration
The abbreviations are stored in `$XDG_DATA_HOME/zsh/abbr`.

- Example
  ```bash
  g=git
  gs=git status
  ga=git add
  gaa=git add --all
  gb=git branch
  gc=git commit
  ```
